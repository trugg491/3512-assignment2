# COMP 3512 Web II Assignment 2

## This website was created for Assignment #2 in COMP3512 at Mount Royal University. The goal of the project is to build a single-page application (SPA) in JavaScript that displays a collection of fashion products with browsing, filtering, and shopping cart functionality.

The site uses HTML, CSS, vanilla JavaScript, and Pico CSS (a CSS-only framework with no JavaScript) to present the data in a clean and organized way. The Home page features categories and trending products, the Browse page lets users filter and sort through 100 products, the Single Product page shows item details with an add-to-cart option, and the Shopping Cart page handles shipping and tax calculations.
